## 7. Comparable Companies Analysis

A relative valuation project that compares companies using key financial metrics and valuation multiples.

The analysis selects a target company and a peer group to evaluate relative valuation using:

- P/E (Price-to-Earnings)
- EV/EBITDA (Enterprise Value to EBITDA)
- EV/Revenue (Enterprise Value to Revenue)

The project calculates peer group statistics, including mean, median, minimum, and maximum valuation multiples.

Median peer multiples are then applied to the target company's financial metrics to estimate implied valuation.

### Concepts Covered

- Comparable Companies Analysis
- Relative Valuation
- Enterprise Value
- Equity Value
- Valuation Multiples
- Peer Group Analysis
- Implied Valuation
- Investment Banking Valuation Fundamentals
