# Financial Analysis with Python

## Overview

This repository contains a collection of beginner-friendly finance and investment analysis projects built using Python. These projects focus on applying financial concepts through programming while developing skills in data analysis, visualization, and financial decision-making.

The repository also includes a project that combines Microsoft Excel and Python to demonstrate how financial data can be analyzed across multiple tools.

---

## Tools & Libraries Used

- Python
- Google Colab
- Pandas
- Matplotlib
- Microsoft Excel

---

## Projects

### 1. Profit & Return Calculator
**File:** `profit_return_calculator.ipynb`
**Notebook:** [Excercise1 :Profit & Return Calculator.ipynb](profit_return_calculator.ipynb)
Calculates investment profit and percentage return based on purchase and selling prices.

**Concepts Covered**
- Investment Return
- Profit Calculation
- Basic Financial Analysis

---

### 2. ROI Calculator
**File:** `roi_calculator.ipynb`

Calculates Return on Investment (ROI) and evaluates investment performance.

**Concepts Covered**
- ROI Analysis
- Investment Evaluation
- Financial Decision-Making

---

### 3. Stock Split Analyzer
**File:** `stock_split_analyzer.ipynb`

Analyzes the impact of stock splits on share quantity and share price.

**Concepts Covered**
- Stock Splits
- Market Mechanics
- Equity Investing

---

### 4. Financial Ratio Analyzer
**File:** `financial_ratio_analyzer.ipynb`

Calculates and analyzes key financial ratios used in company evaluation.

**Concepts Covered**
- Profitability Ratios
- Liquidity Ratios
- Leverage Ratios
- Financial Statement Analysis

---

### 5. Excel to Python Financial Analysis
**Files:**
- `excel_to_python_financial_analysis.ipynb`
- `company_data.xlsx`

A project that combines Microsoft Excel and Python to perform financial analysis and visualization.

**Features**
- Creation of financial datasets in Excel
- Financial ratio calculations
- Excel-based charts and visualizations
- Importing Excel data into Python using Pandas
- Data visualization using Matplotlib

**Concepts Covered**
- Data Analysis
- Financial Ratios
- Data Visualization
- Excel & Python Integration
