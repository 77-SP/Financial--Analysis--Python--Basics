# Investment Analysis using Python

## 📖 Overview

This repository contains beginner-level Python projects focused on investment analysis and financial statement analysis. The exercises are designed to apply basic finance concepts using Python through practical examples and calculations.

The project demonstrates the progression from simple stock return calculations to understanding corporate actions and financial ratio analysis.

---

# 🎯 Objectives

* Understand stock return calculations
* Analyze profit and loss in investments
* Learn the impact of stock splits
* Understand basic financial ratio analysis
* Apply finance concepts using Python
* Build beginner-level analytical skills in finance

---

# 📌 Exercise 1: Single Stock Return Analysis

## Description

Calculates the percentage return of a stock investment and determines whether the investment results in a profit or loss.

## Concepts Covered

* Stock Return Calculation
* Profit & Loss Analysis
* Percentage Returns

## Input

* Buying Price
* Selling Price

## Output

* Return (%)
* Profit or Loss

# 📌 Exercise 2: Investment Analysis with Quantity

## Description

## Concepts Covered

* Selling Price

## Output

* Return (%)
* Profit or Loss (₹)

# 📌 Exercise 3: Stock Split Analyzer

## Description

Analyzes the impact of a stock split on the number of shares, share price, and total investment value.

## Concepts Covered

* Corporate Actions
* Stock Splits
* Share Price Adjustment

## Input

* Company Name
* Number of Shares
* Price per Share
* Stock Split Ratio

## Output

* Shares Before Split
* Shares After Split
* Price Before Split
* Price After Split
* Total Investment Value Before Split
* Total Investment Value After Split

## Example

A 2-for-1 stock split converts:

* 10 shares at ₹1,000 each

Into:

* 20 shares at ₹500 each

The total investment value remains unchanged.

# 📌 Exercise 4: Apple Financial Ratio Analyzer

## Description

Analyzes key financial ratios of Apple Inc. using Python to better understand the company’s profitability, valuation, and shareholder performance.

## Ratios Included

* Price-to-Earnings (P/E) Ratio
* Return on Equity (ROE)
* Profit Margin

## Concepts Covered

* Valuation Analysis
* Profitability Analysis
* Shareholder Performance Analysis
* Financial Statement Analysis

## Purpose

Helps in understanding how financial ratios are used to evaluate a company’s financial performance and market valuation through practical Python-based analysis.

## Output

* Financial Ratio Values
* Basic Financial Performance Insights for Apple Inc.


# 🧠 Key Learnings

* Return (%) measures investment efficiency
* Profit/Loss (₹) shows the actual financial outcome
* Stock splits affect shares and price per share without changing total value
* Financial ratios help evaluate a company’s financial health
* Python can simplify financial calculations and analysis
